MemberJunction Open App · Design System

MJ Forms

One themeable design system, three looks. Every theme is just a set of overrides on MemberJunction's --mj-* design tokens (plus --mjf-* form tokens) — exactly the mechanism MJ dark mode uses, and exactly what a FormStyle stores. Editorial is the default; flip Aurora or Warm live on any page. Same HTML — only tokens change.

The three themes

A theme is ~25 token overrides — no component CSS changes. Brand themes ship the same way (a FormStyle row).
